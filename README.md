좋아하는 말 : 숙제하듯 살지 말고 축제하듯 살자

노인은 손님의 채근에도 아랑곳하지 않으며 방망이를 깎는데 한나절의 시간을 보내고, 방망이 한 벌을 완성하면 굽은 허리를 펴고 무심히 동대문을 바라봅니다. 저는 방망이를 깎는 노인의 태도를 보며, 단순히 돈벌기 위해서 하는 일보다 스스로 의미를 부여하며 즐길 수 있을 때 결과도 잘 나온다고 생각하게 되었습니다.

💬 저는 억지로 하기보단 좋아하고 잘할 수 있는 일을 하면서, 노인처럼 자신의 작업에 꾸준히 정성을 다하는 사람이 되고 싶습니다.
